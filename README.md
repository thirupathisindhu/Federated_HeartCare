# Federated HeartCare AI Web Application

An AI-powered healthcare monitoring system using Federated Learning, Concept Drift Detection, and Model Swapping.

## Features
- Patient Health Input Dashboard
- Federated Learning Simulation
- Concept Drift Detection
- AI Model Adaptation
- Interactive Performance Graphs

## Technologies
- Python
- Flask
- Pandas
- Matplotlib
- HTML / CSS

## Run Locally
1. Activate virtual environment
2. Run:
   python webapp/app.py
3. Open browser:
   http://127.0.0.1:5000
